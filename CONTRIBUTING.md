# Contributing

Thanks for your interest in Freeplay Skills!

## Bug Reports & Issues

If you find a bug or have a question, please [open an issue](https://github.com/freeplayai/freeplay-skills/issues). We actively monitor and respond to issues.

## Pull Requests

Freeplay Skills are primarily maintained by the Freeplay team to ensure consistency with our platform. We're not actively seeking feature contributions, but we do accept:

- **Bug fixes** with clear reproduction steps
- **Documentation improvements**

For feature requests, please open an issue first so we can discuss whether it fits our roadmap.

## Security Issues

Please report security vulnerabilities to security@freeplay.ai rather than opening a public issue.
